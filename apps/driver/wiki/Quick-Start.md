Get Milow up and running in 5 minutes!

## For Users

### 1. Download the App

Visit [GitHub Releases](https://github.com/maninder-mike/milow/releases) and download the latest APK.

### 2. Install

1. Open the downloaded APK
2. Allow installation from unknown sources if prompted
3. Tap "Install"

### 3. Create Account

1. Open Milow
2. Tap "Sign Up"
3. Enter email and password
4. Verify your email

### 4. Start Using

**Add your first trip:**

1. Tap the + button
2. Fill in trip details
3. Save

**View dashboard:**

- See your statistics
- Check recent entries
- View weather and border wait times

That's it! You're ready to track your trips and fuel.

## For Developers

### 1. Prerequisites

```bash
# Verify Flutter installation
flutter doctor
```

### 2. Clone & Setup

```bash
# Clone repository
git clone https://github.com/maninder-mike/milow.git
cd milow

# Install dependencies
flutter pub get

# Setup environment
cp .env.example .env
# Edit .env with your Supabase credentials
```

### 3. Run

```bash
flutter run
```

### 4. Build

```bash
# Android APK
flutter build apk --release

# Android App Bundle
flutter build appbundle --release
```
